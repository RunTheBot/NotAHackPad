# Not A HackPad

Basically a hack pad but with a lot more keys and a lot more LEDs.

# Features
- 100 keys
- OLED Display
- Barrel Jack Power (Pico power is not enough)
- Auto Polarity Correcting Circuit for Barrel Jack
- 100 Neopixels
